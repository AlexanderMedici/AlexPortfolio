# Portfolio Alex

Personal portfolio site built with React and Vite.

## Tech Stack

- React 19 for the UI
- Vite 7 for local dev and production builds
- React Router for client-side routing
- Tailwind CSS 4 with PostCSS and Autoprefixer for styling
- React Three Fiber, Drei, and React Spring for 3D scenes and animation
- EmailJS for contact form email delivery
- React Vertical Timeline for timeline layouts
- ESLint for linting and code quality
